# MVP UX Test Notes

Automated coverage currently checks:

- Desktop landing and login rendering.
- Authenticated dashboard access.
- A 390px-wide dashboard viewport without horizontal overflow.
- Keyboard focus visibility after tab navigation.

Manual checks for each release:

- Use the canvas with touch pan, pinch zoom, selection, and fullscreen.
- Navigate trip sections horizontally on a narrow viewport.
- Complete places, itinerary, and expense forms using only the keyboard.
- Confirm save, realtime, and validation messages are announced or visibly associated with the action.
- Check list/table views remain horizontally scrollable rather than clipping content.
